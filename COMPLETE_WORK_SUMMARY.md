## 🚀 HOW TO USE THE NEW COMPONENTS

### 1. TextAnimate - Text Animation
```tsx
import { TextAnimate } from '@/components/ui'

<TextAnimate 
  animation="blurInUp" 
  by="word" 
  duration={2}
  staggerDelay={0.08}
>
  Your animated text here
</TextAnimate>
```

### 2. GradientText - Gradient Animation
```tsx
import { GradientText } from '@/components/ui'

<GradientText animateGradient={true}>
  Gradient Text
</GradientText>
```

### 3. GlowingText - Glowing Effect
```tsx
import { GlowingText } from '@/components/ui'

<GlowingText color="blue" intensity="medium">
  Glowing Text
</GlowingText>
```

### 4. FloatingElement - Bobbing Effect
```tsx
import { FloatingElement } from '@/components/ui'

<FloatingElement duration={6} distance={20}>
  <div>Floating Content</div>
</FloatingElement>
```

### 5. PremiumTyping - Typewriter
```tsx
import { PremiumTyping } from '@/components/ui'

<PremiumTyping 
  text="Typewriter effect..." 
  speed={40}
/>
```
